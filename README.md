# Professional Scientific Calculator

A robust and visually stunning scientific calculator built with **Vanilla HTML/CSS/JS**.

Unlike traditional web simple calculators, this project features a **Horizontal Wide-Screen Layout** optimized for desktop productivity, a premium **Glassmorphism** design with animated backgrounds, and a dedicated **History Side-Panel**. 

It runs entirely in the browser (Static Site), making it incredibly fast and easy to deploy.
